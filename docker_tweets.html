docker_tweets.news_ingestion_aws module

async docker_tweets.news_ingestion_aws.add_news_article(news_table, news_id, ticker_symbol, item, long_description, keywords, count)[source]

Checks if the scraped news articles pass certain conditions like language check and keyword check and on the basis of those results adds news articles to news table.

Args:

news_table (AWS DynamoDB Table) : “News” Table from DynamoDB news_id (str) : Unique id which is assigned to every scraped article ticker_symbol (str) : ticker name of the company item (response of pygooglenews api) : The object returned of the pygooglenews api long_description (str): content of the article keywords (str) : keywords needed to check their presence in the content of the article count (int) : the number of articles inserted in the table

Returns:

count (int) : the updated number of articles inserted in the table

LastUpdated:

04th July 2023

ChangesMadeInCode:

Added keyword_passed condition in if statement, elif conditions code block for ITC Ltd, State Bank of India, Mahindra & Mahindra Limited.
