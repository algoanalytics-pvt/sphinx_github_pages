docker_stock_ingestion.stock_ingestion.crawl_stock(list_ticker_ids, frequency, database, table)[source]

Main Function for crawling stocks using yfinance library

Parameters

list_ticker_ids: list containing all ticker_ids

The query string which has to be passed to yfinance which will return stocks’ data about all the companies in the column

frequencyint (example = 1 )

Fetches data by interval example if frequency is 1 hour then last one hour data (from current time) will be fetched

database : AWS RDS database in which table are present table : name of table located inside database in which data is to be stored
