docker_debrief.main module

docker_debrief.main.a_intro()[source]

This script fetches data from DynamoDB and S3, processes it, and creates summary JSON files.

Dependencies: - create_json module provides functions like working_check, read_data_from_s3, fetch_data_summary, and fetch_debrief_data. - get_dynamodb_data module provides functions like read_data_dynamodb, read_news_data, and read_tweets_data. - access_file module provides get_secret function to retrieve secret credentials. - helper module provides write_json_file function to write JSON data to files.
